## Getting Started 
Extract the .ZIP folder and open the "Hanoi" folder in Visual Studio Code.
Make sure you have the .NET 10 SDK installed

Open the terminal and ensure that you are inside the project folder "Hanoi"

To run the project: 
- dotnet build
- dotnet run

The program should give you this response *Usage: dotnet run -Recursive [numberOfDisks] or -Iterative [numberOfDisks]*

e.g. *dotnet run -Recursive 3*

This will start an ASCII animation with a total number of three disks, and it will use the Recursive method.
